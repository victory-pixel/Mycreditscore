# Mycreditscore
MyCreditScore is an AI-powered prototype that generates a financial credit score based on mobile money transaction behavior (e.g., MTN MoMo / Airtel Money-style statements).
It demonstrates how artificial intelligence can be used to support financial inclusion by evaluating individuals who do not have traditional banking histories.
# What It Does
MyCreditScore analyzes mobile money transaction data and produces:
A credit score (0–1000)
Behavioral financial analysis
Risk level classification (Low / Medium / High)
Human-readable explanation of the score
How It Works
User inputs or pastes mobile money transaction history
AI analyzes financial patterns:
Income consistency
Spending behavior
Financial stability
The system generates:
Credit score
Risk category
Explanation of reasoning
 Example Input
Plain text
+500,000 MoMo deposit (salary)
-50,000 groceries
-20,000 transport
+300,000 business income
-100,000 rent
📤 Example Output
Credit Score: 740 / 1000
Risk Level: Medium
Explanation: The user shows consistent income patterns with moderate spending. Financial behavior suggests stable repayment potential with minor irregularities.
